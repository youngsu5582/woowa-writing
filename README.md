### 우리의 시작은 돈까스였다

대구에서 25년 생활을 한 나는 

서울에서 살아갈 걱정과 1년 동안의 우테코 생활에 대해서 걱정하며

잠을 설치고 일찍 일어나 계속 불안해 하고 있었다.

<br>

오티 1부가 끝나고 연극 조 크루들과 밥을 먹으라고 하여

어떤 사람들일까 긴장하며 크루들을 찾아갔다.

모두 어색한 순간 적막을 깨는 한마디

```
X : 저희 점심 뭐 먹을까요..?
X : 소울 푸드 돈까스나 먹으러 갈까요??
```

<img src="https://i.imgur.com/rHEccbT.png" width="400" height="400">

그렇다. 우리조는 파이브 가이즈이다.

<br>

언제나 식사 결정 시간이 1분이 넘어본적이 없다.
밥먹는 시간도 30분이 넘어본적 없다.

우테코 오기까지 1시간 반동안의 걱정은
남고 와 공대에서 느낀 익숙한 맛에 돈까스와 함께 사라졌다.

<br>

모두 금쪽이들이라 솔직히 주말에 우테코 생활이 아쉬울 때도 있다.

우리 조원중 한명이 술자리와 유강스( 유연성 강화 스터디 ) 에서 맨날 하는 말이 있다

### 코드 짜는 금쪽이

내 유연성 강화는 `상대방이 나와 다른 의견일 때 틀림이 아닌 다름을 인정한다.` 였다

어릴 때부터 맞다고 생각한 부분이나 한 부분에 꽂히면 누가 뭐라해도 마이 웨이였다.

그리고 , 페어 프로그래밍 하면서도 `그래 너 의견 오케이 근데 내 말이 맞아`  같은 놀부 심보는 끊임없이 발동했다.

( 미안해 나의 X들 ,,, )

<img src="https://i.imgur.com/YOk7J0p.png" width="600" height="400">

<br>

"아 그거 그렇게 하는거 아닌데,,"

"코드 그렇게 짜면 더러워서 불편한데;;"

라는 속마음과 함께 페어에게 신나게 내 의견을 주장했다.

서로 그렇게 얘기를 해나가다 의견이 계속 굳혀지지 않고 ...

```
X : 그러면 그냥 일단 조이썬의 방식대로 하고 리뷰어한테 피드백 받을까요??
나 : 아 그러면 , 일단 제 방식대로 할까요??
( 당연히 이 방식이 더 좋으니 , 이렇게 해야지 😈😈 )
```

<br>

결과는 Fail 이였다.

내가 자신있게 밀고 좋다고 설명한 방식은 리뷰어 한테 별로라는 평가를 받았다.

들이박을까 라고 생각했지만 포기했다 ( 그들이 나보다 더 잘하는걸 알기에,, )

<br>
페어한테 미안함을 느꼈다

내가 자신 있게 어필하고 , 내 방식대로 코드를 작성하자고 했으니까 더 그랬다.. 😭😭

그래서 강화 주제는 속죄의 마음으로 `틀림이 아님 다름을 인정한다` 가 되었다

### 지금은..? 

지금도 자기 주장 강한 금쪽이다
부정하지 않는다

<br>

하지만 이제는 왜 그렇게 주장하는지 + 왜 그렇게 하기 싫어 하는지 설명하고

해당 설명을 토대로 좀 더 토론을 다시한다

"니 말은 틀리고 내 말은 맞아" 가 아닌 두 가지 주장에 대해서 생각해보고

<br>

```
내가 생각했을 때는

이 방식대로 하면 나중에 코드가 이렇게 저렇게 ...
이 방식대로 하면 어쩌고 저쩌고 ...
될 거 같은데

의견에 어떻게 생각하는지 말해주고
이제 좀 더 얘기해 보면 될 거 같은데??
```

찬반 투표가 아닌 지성있는 토론을 이끌어 나아가려고 노력한다

1. 상대가 답답해서 포기하고 니 의견이 맞다고 한다

3. "반박시 니 말이 맞음" 대화 원천 봉쇄가 가능하다
